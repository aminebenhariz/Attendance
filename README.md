## Install

Install `Attendance` using Composer.

```
$ composer require aminebenhariz/attendance
```

## Requirements

The following versions of PHP are supported by this version.

+ PHP 5.4
+ PHP 5.5
+ PHP 5.6
+ PHP 7.0
+ HHVM
